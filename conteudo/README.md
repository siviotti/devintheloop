# Conteúdo da Abordagem Dev In The Loop

Dev In The Loop é uma abordagem heurística para tomada de decisão no uso de IA Coding em projetos de desenvolvimento de software e, dessa forma, provê um conjunto de heurísticas que são as ferramentas que axiliam nestas decisões. Contudo, a abordagem também apresenta algumas práticas que ajudam no processo de desenvolvimento de software. Assim, o conteúdo está dividido em dois grupos:

1. [Heurísticas](/conteudo/heuristica/README.md) - Ferramentas que ajudam na tomada de decisão sobre o uso de IA Coding, especialmente para responder a pergunta "Quando e como entrar no Loop?"
2. [Práticas](/conteudo/pratica/README.md) - Ferramentas que ajudam na construção e/ou manutenção do software propriamente dito, gerando entregáveis convencionais do processo de desenvolvimento ou artefatos apresentados concebidos especialmente para a prática em questão. As práticas que geram um ou mais artefatos entregáveis são denominadas "Práticas de Entrega" e as práticas que não geram entregáveis são denominadas "Práticas de Suporte".
